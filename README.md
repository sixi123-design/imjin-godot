# 불멸의 성채: 임진왜란 — Godot 4 버전

They Are Billions 스타일의 조선 수성전. 아이소메트릭 뷰 · 워포그 · 밤낮 주기와 동적 조명 ·
일꾼 건설/농사/벌목/채광 · 건물 개축 · 정예 병사 · TAB식 호드 공세 · 미니맵.

## 실행 방법

1. https://godotengine.org/download 에서 Godot 4.2 이상 다운로드 (.NET 불필요)
2. 프로젝트 매니저 → 가져오기(Import) → 이 폴더의 `project.godot`
3. F5 실행

## 조작법

- 하단 [건설] / [병력] 탭: 생산(벌목장·농경지·철광·민가) | 방어·군사(성벽·탑·훈련소·공방) 그룹 정렬, 비용은 재화 아이콘으로 표시 · 상단 [전체선택] = 전군 선택
- 건물 버튼 선택 후 지도 클릭 = 건설 (우클릭/ESC 취소) · 안개 속에는 건설 불가
- **철광은 잿빛 광맥 타일 위에만** 건설 가능 — 광맥은 맵에 7군집, 첫 군집만 시작 시야권
- 지어진 건물 클릭 = 선택 → 우하단 패널에서 개축
  - **목책→석벽**(내구 170→380) · 초가집→기와집(병력+8) · **농경지→논(2×2, 식량 4.2/초, 주변 공간 필요)**
  - 궁수탑→수성망루(공격↑) · 훈련소→훈련도감(정예 해금)
- **목책·석벽은 아군 병사도 통과 불가 — 출입 통로를 남겨 두어라** · 석벽 직접 건설 가능
- **농경지·논·벌목장·훈련소·훈련도감은 2×2 대형 건물**
- 병력 탭: 궁수/창병, 훈련도감 보유 시 편전수/갑사 · 시작 병력은 궁수 1
- 병사 클릭·드래그 선택 → 우클릭 이동
- 우상단 미니맵: 지형/건물/아군(녹)/적(적) 표시, 클릭·드래그로 화면 이동
- **WASD** / 휠클릭 드래그 / 두손가락 팬 = 화면 이동 · 휠/핀치 = 줌 · Space 일시정지

## 시스템

- 맵 64×64 정사각 타일, 워포그: 미탐사 완전 흑색(맵 외곽 포함), 탐사 경계는 2단 페이드
- 지형: 철 광맥(철광 전용) · **절벽(이동·건설 불가)** · **진흙(이동 가능·건설 불가)** — 적도 절벽을 우회
- 성벽은 단일 이미지(오토타일 아님) — 목책(개축 가능)과 석벽 2단계
- 일꾼은 조선 백성 복장(백의·상투·망건), 작업별 구분: 건설(흰 머릿수건)/농사(패랭이)/벌목(등거리)/채광(머리띠+소쿠리)
- 공세는 새벽에 도착, 방향은 45초 전 대형 경고(가장자리 붉은 띠 + 화살표 + 카운트다운 + 미니맵 표시)
- 총 8차 공세: 차수마다 적 체력/공격력/수가 증가 · 마지막은 사방 총공세
- 신규 적: **왜장**(2배 덩치, 주변 적 공·이속 버프 오라) · **폭탄병**(접촉 시 자폭, 건물·병사 광역 피해)
- **공방**(연구 건물): 건물 공격/방어, 유닛 공격/방어 각 3레벨 연구
- 정예 병사(편전수·갑사): 비용 10배·인구 2 대신 강력 — 훈련도감 필요
- 생산 건물 위로 생산량이 주기적으로 표시(+8 식량 등), 작업 중 일꾼은 건물 앞에 표시
- **밸런스 튜닝: `balance.cfg`** — 건물/유닛/적/공세/전역 수치 전부 이 파일에서 수정 (한글 주석 포함). 오타·잘못된 항목은 무시되고 화면 좌하단에 경고 표시. main.gd 상단 값은 cfg 로드 실패 시의 기본값

## 그래픽

`art/` 폴더의 조선풍 4배 해상도 스프라이트 (프로시저럴 생성 — art_src/gen1~6.py로 재생성)
2×2 건물 5종 + 일꾼 4종 + 왜장/폭탄병 + 광맥/진흙/절벽 타일 + 재화 아이콘 3종 포함
